# Kopiera objekt mellan sandlådor

Lär dig hur du kopierar objekt mellan sandlådor i Experience Platform med hjälp av paket. Kopiera enkelt scheman, datauppsättningar, resor med mera mellan sandlådor för att stödja releaseprocesser och driftsättningar för flera regioner/varumärken. Mer detaljerad information finns i dokumentationen för [sandlådeverktyget](https://experienceleague.adobe.com/docs/experience-platform/sandbox/ui/sandbox-tooling.html).
