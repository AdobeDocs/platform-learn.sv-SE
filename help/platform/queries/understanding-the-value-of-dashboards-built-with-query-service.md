# Förstå värdet på kontrollpaneler som skapats med frågetjänsten

Med Adobe Experience Platform kan varumärken importera, lagra, strukturera och hämta in alla lagrade datauppsättningar, inklusive beteenden, CRM, butiksdata med mera. Med hjälp av Experience Platform Query Service kan varumärken ställa frågor om dessa datauppsättningar och besvara specifika frågor om verksamheten och sedan börja generera slagkraftiga insikter. I den här videon visas värdet av att bygga instrumentpaneler i Business Intelligence-verktyg (BI) med hjälp av Query Service. Mer information finns i [dokumentationen för frågetjänsten](https://experienceleague.adobe.com/docs/experience-platform/query/home.html?lang=sv).
