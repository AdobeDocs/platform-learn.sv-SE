# Adobe Experience Platform Mobile SDK - översikt

Lär dig om hur Adobe Experience Platform Mobile SDK kan skapa ett heltäckande engagemang i dina mobilapplikationer. Med Platform Mobile SDK kan du inhämta, analysera, engagera och optimera. Högpresterande, modulärt, utbyggbart och integrerat med datainsamling. Datainsamlingens sofistikerade regelmotor, behörigheter och publiceringsarbetsflöden hjälper dig att effektivt implementera Adobe Analytics-, Campaign-, Audience Manager, Target- och tredjepartslösningar i dina mobila appar. Mer information finns på [SDK-dokumentation för plattformsmobil](https://developer.adobe.com/client-sdks/documentation/).
