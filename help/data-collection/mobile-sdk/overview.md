# Adobe Experience Platform Mobile SDK - översikt

Lär dig om hur Adobe Experience Platform Mobile SDK kan skapa ett heltäckande engagemang i dina mobilapplikationer. Med Platform Mobile SDK kan du inhämta, analysera, engagera och optimera. Högpresterande, modulärt, utbyggbart och integrerat med datainsamling. Datainsamlingens sofistikerade regelmotor, behörigheter och publiceringsflöden hjälper er att effektivt implementera Adobe Analytics-, Campaign-, Audience Manager-, Target- och tredjepartslösningar i era mobilappar. Mer information finns i [dokumentationen för mobila SDK-plattformar](https://developer.adobe.com/client-sdks/documentation/).
